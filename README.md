## 说明
```
一个仿低代码平台，支持文本，矩形，图片，视频，表格等组件的拖拽生成以及编辑组件的属性，支持生成图片以及生成JSON数据。
```


## 技术栈

+ Vue
+ Vuex
+ ElementUi
+ Echarts
+ Vue-draageble-resize
+ vue-color
+ file-saver



## 数据支持

+ 静态数据
+ API接口



## 初始化

```
yarn  或 npm install
```

### 开发

```
yarn dev 或 npm run dev
```

### 打包

```
yarn build 或 npm run build
```
